Explicacin

  1. train_test_split(df, test_size=0.3) divide el DataFrame en un 70% para entrenamiento y un 30% para prueba.
  2. TfidfVectorizer() convierte el texto (columna 'news') en vectores numricos usando TF-IDF.
  3. fit_transform() y transform() generan la matriz de caractersticas para entrenamiento y prueba.
  4. toarray() convierte las matrices dispersas a arrays NumPy densos.
  5. Y_train y Y_test son las etiquetas objetivo.
  6. K es el nmero total de clases (asumiendo clases de 0 a K1).
  7. D es la dimensin de entrada, es decir, el nmero de caractersticas por muestra.

Modelo

Construccin del Modelo

In[23]:
# Capa de entrada
i = Input(shape=(D,))
# Capa Densa
x = Dense(300, activation='relu')(i)
#Capa de salida Softmax
x = Dense(K)(x)

#Creacin del modelo
modelo = Model(i, x)

Entrenamos el Modelo

In[]:
r = modelo.fit(
    X_train,
    Y_train,
    validation_data= (X_test, Y_test),
    epochs= 100,
    batch_size= 12   #Total de datos / 12
)

Explicacin

Arquitectura del modelo:

  • Input(shape=(D,)): entrada de dimensin D (viene de la matriz TF-IDF).
  • Dense(300, activation='relu'): una capa totalmente conectada con 300 neuronas y activacin ReLU.
  • Dense(K): capa de salida con K neuronas (una por clase). No lleva activacin porque usamos from_logits=True en la prdida.

Compilacin:

  • Loss: SparseCategoricalCrossentropy(from_logits=True) se usa porque las etiquetas estn codificadas como enteros y la salida an no pasa por softmax.
  • Optimizer: 'adam' para una optimizacin eficiente.
  • Mtrica: 'accuracy' para evaluar el desempeo.

Entrenamiento:

  • Entrena durante 100 pocas, usando lotes de tamao 12.
  • Se valida en cada poca usando X_test y Y_test.

# Ejemplo de una nueva noticia
nueva_noticia = ["El presidente anunci nuevas medidas \
                    econmicas para enfrentar la inflacin."]

# Transformar la noticia con el mismo vectorizador usado en entrenamiento
X_nueva = vectorizer.transform(nueva_noticia).toarray()

# Obtener la prediccin (logits)
Predict = modelo.predict(X_nueva)

# Aplicar softmax para convertir P en probabilidades
probs = tf.nn.softmax(Predict).numpy()

# Obtener la clase predicha
clase_predicha = np.argmax(probs)

# Obtener el mapeo de cdigos a etiquetas
category_mapping = dict(enumerate(df['Type'].astype('category').cat.categories))

# Creamos diccionario para almacenar la probabilidad de cada categora
probs_dict = {
    category_mapping[i]: float(prob)
    for i, prob in enumerate(probs[0])
}
display(HTML(f"<h3><b>Clase predicha para la noticia:</b>\
                {category_mapping[clase_predicha]}</h3>"))

#Salida en pandas
# Crear DataFrame con categoras y probabilidades
df_probs = pd.DataFrame({
    'Categora': list(probs_dict.keys()),
    'Probabilidad': list(probs_dict.values())
})

# Ordenar por probabilidad descendente
df_probs = df_probs.sort_values(by='Probabilidad', ascending=False).reset_index(drop=True)
